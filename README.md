DataSight CRM Web Applicatie
MVP ontwikkeld in samenwerking met LiQ

Webapplicatie die een centraal overzicht biedt van hun data, aangevuld met een geavanceerd forecasting-algoritme.

Installatie

1. pip install -r requirements.txt

2. Configureer .env met database credentials

3. python run.py
